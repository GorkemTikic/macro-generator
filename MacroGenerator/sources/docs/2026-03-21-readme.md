# README.md summary

## Aim

FD Macro Generator: High precision premium support automation tool for futures support agents. It produces data-backed macro answers to common user questions such as Slippage, Stop Market trigger, Funding Rate, etc., with 1m OHLC data pulled from the Binance Futures API.

## Architecture (summary)

**Stateless Macro Engine** — Collects UI input, fetches price via [[entities/pricing-ts]], passes it to pure format functions (macros).

**Stack**: React 18 + Vite + TypeScript, vanilla CSS (Premium Dark Mode), GitHub Pages deploy.
